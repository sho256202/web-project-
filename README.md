توثيق المشروع النهائي لمساق تطوير الويب

## وصف المشروع
مشروع GlamourBeauty هو عبارة عن واجهة مستخدم لمتجر إلكتروني متكامل ومتجاوب مخصص لعرض مستحضرات التجميل والعناية. تم بناء وتطوير هذا الموقع بالاعتماد على تقنيات الويب الأساسية لتقديم تجربة مستخدم سلسة وتصميم عصري يتناسب مع كافة الشاشات وأجهزة الهاتف المحمول.


## فكرة المشروع
تتمثل فكرة المشروع في تصميم واجهة متجر تفاعلية تحقق المعايير البرمجية التالية:
1. بناء هيكل تنظيمي واضح ومدروس للموقع باستخدام وسوم لغة HTML5 الدلالية لضمان تنظيم الكود البرمجي وسهولة مراجعته وتطويره.
2. تنسيق العناصر وتوزيع المساحات والخطوط والألوان بدقة واحترافية عالية باستخدام لغة CSS3، والاعتماد على تقنيات التوزيع الحديثة مثل Flexbox و CSS Grid لضمان مرونة التصميم وتجاوبه الكامل.
3. إضافة ميزات تفاعلية حيوية داخل الموقع باستخدام لغة JavaScript، حيث تم برمجة النوافذ المنبثقة المخصصة لتعرض رسائل تفاعلية فريدة عند الضغط على أزرار المتجر، بدلاً من استخدام رسائل التنبيه التقليدية للمتصفح.

---

## الأدوات والتقنيات المستخدمة
تم إنجاز هذا المشروع بالاعتماد على الأدوات والبرمجيات التالية:
* لغة هيكلة محتوى الويب (HTML5).
* لغة تنسيق وتصميم صفحات الويب (CSS3).
* لغة البرمجة التفاعلية (JavaScript).
* مكتبة الأيقونات الخارجية (Font Awesome) لاستدعاء أيقونات التواصل والأزرار.
* خط (Montserrat) المستدعى من خطوط جوجل لإضفاء طابع رسمي وعصري على النصوص.
* محرر الأكواد (Visual Studio Code) لبيئة التطوير وكتابة الشفرات البرمجية.
* منصة (GitHub) لإدارة النسخ ورفع التعديلات واستضافة الموقع عبر خدمة GitHub Pages.
